# option-9-java

## Commiting/Pushing 

first make sure you remove the build folder

1. add files to git
use command 
```bash
git add *
```

2. Commit to local repo
```bash
git commit -m "{commit message}"
```

3. Upload to github
```bash
git push --set-upstream https://github.com/open-circuits-robotics/option-9-java.git main
```
